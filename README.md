# PDF Manager

Python (PyQt6, PyMuPDF) 기반의 빠르고 가벼운 로컬 PDF 관리 및 편집 도구입니다. 복잡한 설치 없이 제공되는 **설치 프로그램(`Setup.exe`)** 하나로 간편하게 윈도우 환경에 설치하고 사용할 수 있도록 개발되었습니다.

## 주요 기능
*   **PDF 뷰어 및 페이지 관리:** 페이지 이동, 확대/축소, 썸네일 탐색
*   **PDF 병합 및 추출:** 여러 PDF 파일을 하나로 합치거나 특정 페이지만 추출하여 새 PDF로 저장
*   **이미지 ↔ PDF 상호 변환:** 폴더 내 이미지 파일들을 PDF로 묶거나, 반대로 PDF의 각 페이지를 고화질 이미지(PNG, JPG 등)로 일괄 추출
*   **PDF 텍스트 및 주석(Annotation) 편집:** 현재 페이지에 텍스트 팝업 노트(주석)를 추가/삭제하고, 간단한 텍스트 또는 이미지를 삽입하는 기능 보유
*   **텍스트 추출:** 현재 페이지에 포함된 텍스트들을 긁어와 TXT 형태로 저장
*   **고급 PDF 최적화 (Acrobat 스타일):** 용량을 획기적으로 줄이거나 품질을 지정하기 위한 맞춤형 이미지 다운샘플링 및 압축 지원
    *   컬러, 회색명암 해상도(>225 ppi) 시 150 ppi로 바이큐빅 기반 축소 후 JPEG 압축
    *   단색(흑백) 해상도(>450 ppi) 시 300 ppi로 축소 후 TIFF(CCITT Group 4) 압축
*   **보안 및 암호 기능:** PDF 파일에 조회용 비밀번호, 편집용 비밀번호, 권한(복사, 인쇄 등) 제약 설정 후 저장 기능 보유
*   **OS 파일 연결 연동:** 윈도우 환경에서 PDF 파일을 더블클릭할 시 이 앱으로 바로 열리게 됨

## 설치 및 실행 방법

번거로운 파이썬 환경 설정이나 명령어 입력 없이, 빌드된 **설치 파일**을 통해 프로그램을 바로 사용할 수 있습니다.

1.  이 리포지토리의 파일 목록에서 다음 경로로 이동합니다.
    *   `src/dist/PDF_Manager_Setup.exe`
2.  해당 **[설치 파일 다운로드](src/dist/PDF_Manager_Setup.exe)**를 클릭하여 PC에 내려받습니다.
3.  다운로드받은 `PDF_Manager_Setup.exe` 파일을 **실행**합니다.
4.  화면의 안내에 따라 설치를 완료하면, 바탕 화면에 생성된 바로가기 아이콘이나 시작 프로그램 메뉴에서 앱을 편리하게 실행하실 수 있습니다!

## 활용된 핵심 라이브러리
*   `PyQt6` (GUI)
*   `PyMuPDF` / `fitz` (PDF 엔진 처리 로직)
*   `Pillow` (이미지 최적화 처리)
