# PDF Manager

파이썬을 활용한 PDF 파일 관리, 텍스트 추출, 병합 등을 처리하는 프로젝트입니다.

## 폴더 구조
- `src/`: 실제 PDF 처리를 담당하는 파이썬 소스 코드가 위치합니다.
- `pdfs/`: 작업할 원본 PDF 파일과 처리된 결과물 PDF 파일을 보관하는 폴더입니다.
- `tests/`: 기능 테스트를 위한 코드가 위치합니다.

## 시작하기
1. 가상환경(venv 등)을 생성하고 활성화할 것을 권장합니다.
2. `pip install -r requirements.txt` 명령어를 통해 필요한 라이브러리를 설치합니다.
3. `src/main.py`를 실행하여 프로그램의 기본 동작을 확인합니다.
